## Décisions capturées

- **UX-First rule always-loaded** (pas path-triggered) — cohérence
  avec vision-first + creative-ambition. +2.5 KB par prompt accepté.
- **Catalogue interactions/ séparé de mechanics/** — distinction
  motion visuelle ≠ comportement. Slug différent = filtrage clair par
  la règle.
- **6 patterns au départ, pas 20** — discipline anti-slop. Grow-it
  quand pattern réel émerge, pas théorique.
- **Emerging inputs (gesture/voice/gaze/haptic/audio) différés** en
  Phase future avec tagging maturity explicit (research / experimental /
  production). Aujourd'hui = pas shippable web en 2026, pas d'honnêteté
  sans le dire.
