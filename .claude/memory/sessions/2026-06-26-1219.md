# Session — Bespoke "The Hidden Shore" private-journey pitch page

## What changed

New standalone presentation page for a one-off client pitch (a private
birthday journey along the Algarve coast). Branch
`feat/hidden-shore-presentation`, commit `e9473e2`.

- **Route** `/the-hidden-shore` — top-level, outside the locale tree, no
  layout chrome, `noindex`. Same standalone pattern as `/presentation`
  ([pattern](../patterns/2026-06-17-standalone-public-page-outside-localeprovider.md)).
- `src/pages/TheHiddenShore.tsx` — dark hero → manifesto → 8 editorial
  chapters (2 photo plates woven in) → Option A/B departure timeline

## Decisions taken (autonomously)

- **Option A / Option B = the two departure schedules** (11:00 vs 14:00).
  The pasted brief describes a single experience with dual morning times;
  the toggle swaps only those. Flagged the interpretation to the owner.
- **English-only, not i18n/Sanity.** Bespoke one-off document for an
  anglophone recipient. Wiring FR/ES for ~200 lines of pitch copy would
  be waste. `validate:i18n` still passes (no hardcoded FR). Departs from
  the usual "never hardcode" rule deliberately — noted here.
- **Photo slots** = `public/images/hidden-shore-yacht.jpg` +
  `hidden-shore-beach.jpg`. Until present, a clean mono placeholder
  shows; dropping the files needs no code change.
