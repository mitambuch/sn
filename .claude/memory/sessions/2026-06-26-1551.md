# Session — The Hidden Shore pitch page: rebuilt, video cover, shipped live

Continuation of [the morning entry](2026-06-26-1219.md). The page was
iterated on owner feedback and deployed to production.

## What shipped

Live at **https://saw-next.ch/the-hidden-shore** (Netlify-served; Vercel
also deploys the repo). Merged via PR #13.

- **Layout rebuilt** (owner: first pass was an endless scroll). Now a
  short, tap-to-open page: cinematic cover → at-a-glance facts → collapsed
  accordions (story in 8 chapters, day in 4 acts, included by category).
  No bullet lists (hairline-separated), lucide pictos, reuses the
  `Accordion` atom. Owner picked the accordion model from a 3-option
  AskUserQuestion.
- **Cover background video** (`public/video/portimao.mp4`), self-hosted
  like the landing clips. Owner supplied 25.9 MB / 1080p → compressed to
  4.3 MB / 720p with ffmpeg two-pass (installed via winget this session).
- **Mobile A/B switch** pinned to the bottom of the screen (thumb reach),
  visible only while The Day is in view (IntersectionObserver); inline
  toggle on desktop. Buttons later restyled to a stacked, compact label.
- Photos dropped (owner replaced them with the video).

## Decisions / notes

- English-only, bespoke, not in the i18n/Sanity stack. noindex.
- Could not local-merge to main (CLI guardrail) → used a PR + `gh pr merge`
  on owner approval, which is the cleaner flow anyway.
- Fixed a pre-existing invalid `#frontend` tag on the GROQ friction entry
  (removed it; `#sanity`+`#i18n` already cover it) — unblocks
  `memory:index:check`. See also [[powershell-heredoc-double-quotes]].

## Caveat

GitHub CI is red independent of the deploy: the bundle-size check counts
`public/video/*` (~13 MB across 3 clips) against a 3500 KB threshold —
pre-existing since the landing clips landed (2026-06-17), worsened by this
clip. The Netlify/Vercel build (`pnpm build`) is unaffected and passes.
Worth a follow-up: raise the bundle threshold or exclude video from the
measure.
