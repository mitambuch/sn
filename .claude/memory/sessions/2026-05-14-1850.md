  granted to authenticated only. Fixes the single-use security gap.

- **AuthContext.confirmInvitationRedemption** action that calls the
  RPC + maps the json envelope to user-facing FR messages.

  now read from Supabase via `useInquiriesUser`.

  `useInvitationsAdmin`.

  AdminShareCodes placeholder + empty state, ExamplePage <title>).
- **E2E** : 5 specs covering /admin/{invitations,inquiries,users} +
