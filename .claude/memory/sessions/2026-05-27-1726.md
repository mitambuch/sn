## Friction rencontrée

1. **Env Vite pas dans le bundle au 1er build** — owner avait set
   env vars APRÈS le deploy initial. Vite inline `VITE_*` à BUILD time,
   donc rebuild "Clear cache and deploy site" requis. Bundle a alors
   reçu URL + ANON_KEY → `hasSupabase=true`.

2. **Resend sandbox restriction** — sender `onboarding@resend.dev`
   ne peut envoyer qu'à l'email du compte Resend (sécurité anti-spam).
   Quand owner a essayé OPERATOR_EMAIL=info@saw-next.ch → rejet
   Resend "Testing domain restriction". Solution immédiate : flip
   OPERATOR_EMAIL temporairement vers perso Resend signup. Solution
   définitive (PUNCH LIST 1) : verify saw-next.ch sur Resend.

3. **Vault secret duplicate** — RESEND_API_KEY existait déjà
   (probablement créé lors de session 2026-05-11). Update via SQL
   `update vault.secrets set secret = ... where name = ...` plutôt
   que insert.

**État actuel** : emails partent de `onboarding@resend.dev` vers le
perso Resend signup. L'opérateur ne reçoit donc PAS encore à
`info@saw-next.ch`.

**À faire** :
1. Resend Dashboard → Domains → **Add Domain** → `saw-next.ch`
2. Resend donne 3 records DNS (sur sous-domaine `send.saw-next.ch`
   pour éviter conflit avec Infomaniak mail) :
   - `MX send.saw-next.ch` → `feedback-smtp.eu-west-1.amazonses.com`
     priority 10
   - `TXT send.saw-next.ch` → `v=spf1 include:amazonses.com ~all`
   - `TXT resend._domainkey.saw-next.ch` → long DKIM key
3. Ajouter sur Infomaniak Manager → DNS de saw-next.ch (additif,
   PAS toucher SPF apex Infomaniak ni MX existant)
4. Attendre verification Resend (1-30 min)
5. Une fois "Verified" sur Resend Dashboard :
   - Run [supabase/migrations/0013_email_sender_saw_next_ch.sql](
     ../../supabase/migrations/0013_email_sender_saw_next_ch.sql)
     dans Supabase SQL Editor (recreate les 2 trigger functions avec
     `from: noreply@saw-next.ch`)

     'info@saw-next.ch' where name = 'OPERATOR_EMAIL'`
6. Submit test sur saw-next.ch → confirmer email arrive à
   `info@saw-next.ch` (status "Delivered" sur Resend Logs)
