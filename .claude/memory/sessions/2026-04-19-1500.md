### Restructure Playground.tsx

- 2 zones visuellement séparées par `<ZoneBanner>` : CORE + GRAPHIC.
- Chaque section désormais sur sa propre ligne (1-col stack, plus de
  2-col wrappers).
- Section numbers renumérotés séquentiellement (01-14 CORE, 15-24
  GRAPHIC).

### Dette résolue

- **Husky merge commit bug** : `.husky/pre-commit` détecte maintenant
  `.git/MERGE_HEAD` et autorise les merge commits `--no-ff` sur main.
  Friction `frictions/2026-04-19-husky-blocks-merge-commits.md` close.
- **Marquee keyframes manquantes** (ticker figé) — fixé dès le début.
- **Canonical Tailwind warnings** : 6+ occurrences `max-w-[1600px]`
  → `max-w-400`, `bg-gradient-*` → `bg-linear-*`.
- **Double-scroll bug** : useBodyScrollLock wiré dans Fullscreen,
  Corner, CommandPillK palette, MegaDropdown mobile.

### Totaux chiffrés

`src/components/ui/` préservés (atoms). Toute la librairie utilise des
tokens CSS (`var(--color-*)` ou classes `bg-accent` etc.), compatible
light + dark mode automatiquement.
