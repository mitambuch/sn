# UI Styles — catalogue vocabulary

## Usage

~80 styles UI extraits de UI UX Pro Max, triés en **3 tiers** :

- **Tier 1 — Signature / haute ambition** : ce qu'on veut proposer
  par défaut sur un site "putain de ouff". Direction assumée, aesthetic
  à caractère, vocabulaire moderne.
- **Tier 2 — Vocabulary** : utile comme nom quand un brief arrive
  ("je veux du brutalism" → je sais ce que ça veut dire). Peut servir
  de support mais rarement la star.
- **Tier 3 — Slop-warn** : styles que l'AI générative default-produit
  quand elle n'ose rien. Éviter par défaut. Lister ici pour
  reconnaissance, pas pour adoption.

Un style peut être Tier 1 **sur un projet** (brutalism pour un label
musique) et Tier 2 **sur un autre** (brutalism pour une banque →
bruit). Le tier est indicatif, pas absolu.

## Tier 3 — Slop-warn (default AI output, éviter sans brief explicite)

Ces styles ne sont PAS mauvais — ils sont ce que GPT/Gemini/Claude
default-produit quand tu dis "luxury" ou "wellness". Les garder en
contexte uniquement si un brief explicite les demande et qu'on sait
pourquoi. Sinon les éviter au profit du Tier 1.

- **Soft UI Evolution + palette Soft Pink + Cormorant Garamond + Gold CTA**
  → l'exemple README de uipro lui-même. Cliché wellness-spa-luxury.
- **Hero centré + fade-in + gradient violet-rose pastel** → landing
  2021 par défaut.
- **Card grid 3-col + shadow subtle + "Our Services"** → corporate
  bootstrap.
- **Trust & Authority** (tel quel, sans remix) → "As seen on" + logo
  cloud + testimonial avatar → corporate 2015.
- **Minimalist Monochrome + Inter body + centered layout** → réponse
  safe à "minimal" qui ne dit rien.

Voir `anti-patterns/ai-slop-luxury-spa.md` (Phase 2) pour le détail.

## Comment utiliser cette fiche en VISION

Exemple — brief "concept store parisien, 4 valeurs à mettre en avant,
galerie art" :

```
VISION
- Intent : 4 valeurs comme œuvres en mouvement dans une galerie immersive
- Refs : uipro:Parallax Storytelling + uipro:Kinetic Typography +
        mechanics/rotating-3d-words + references/bruno-simon
- Mechanics : rotating-3d-words, scroll-driven-3d, magnetic-hover
- Style primaire : Exaggerated Minimalism (typo oversized galerie)
- Style signature : Kinetic Typography (les mots respirent, mutent)
- Keep : cohérence brand voice client
- Drop : carousel classique, fade simple
- Verdict attendu : go | refs missing | repropose
```
